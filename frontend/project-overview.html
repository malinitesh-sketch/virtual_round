How to use it

Step 1: Start backend

Run your backend from the `backend` folder. Confirm `http://localhost:5000/api/health` returns a success message.

Step 2: Open frontend

Open the frontend page in your browser. Use the overview page for a quick summary, then open the interactive dashboard or project pages.

Step 3: Use Blackbox

Open `BLACKBOX_GUIDE.md`, install Blackbox AI, and write descriptive comments in backend files to generate code.

Step 4: Extend features

Add database models, authentication, trip management, activity scheduling, and user profile updates using your current project structure.
