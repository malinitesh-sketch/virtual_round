🚀 Git & VS Code Quick Guide

Odoo Hackathon ma code manage karva mate niche na steps follow karo.

1. VS Code Connection (Setup)

VS Code ma Terminal kholo (Ctrl + `) ane aa command nakho:

git config --global user.name "Tamaru Naam"
git config --global user.email "tamaru@email.com"

2. Repository Clone Karvi

Project ne tamara PC ma lavva mate:

git clone [REPO_URL_HERE]

3. Navi Branch Banavvi (Most Important)

Main code ne nuksan na thay etle potani branch ma kaam karo:

git checkout -b feature-tamaru-naam

*Aa command thi navi branch banshe ane tame ema shift thai jasho.

4. Code Push Kevireete Karvo?

Kaam thai gaya pachi code online mukva mate:

git add .
git commit -m "Briefly explain what you changed"
git push origin feature-tamaru-naam

5. Code Merge Kevireete Karvo?

Potano code main branch sathe jodva mate:

  1. GitHub par jao.
  2. "Compare & pull request" button par click karo.
  3. Team leader (Nitesh) check kari ne "Merge pull request" dabshe.
⚠️ Dhyan Rakho: Kaam sharu karta pehla hamesha git pull origin main karo jethi badha no code up-to-date rahe.
